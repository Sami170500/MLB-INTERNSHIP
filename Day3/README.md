# Student Record Management System

## Description
This is a Python-based Student Record Management System that stores student data in a JSON file. The project allows users to manage student records through a simple menu-driven interface.

## Features
- Add Student Record
- View All Students
- Search Student by Roll Number
- Update Student Details
- Delete Student Record
- Store Data Permanently Using JSON
- Prevent Duplicate Roll Numbers

## Technologies Used
- Python
- JSON

## Files
- `Student_Record_Management_System.py` - Main Python program
- `students.json` - Stores student records
- `README.md` - Project documentation

## Sample Student Record

```json
{
    "Name": "Sami",
    "Rollno": "B29952",
    "Age": 21,
    "Course": "BSCS"
}
```

## How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Run the Python file:

```bash
python Student_Record_Management_System.py
```
